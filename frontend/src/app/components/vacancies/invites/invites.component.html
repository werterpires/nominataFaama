Atenção

Se esse é o primeiro chamado que faz nesse sistema, considere os itens abaixo:

  1. Antes de chegar a um estudante, o convite passará pela coordenação do Salt ou pelo ministerial para ser aprovado.
  2. A vaga não poderá ser editada após a aprovação ou reprovação de pelo menos um convite. Esse é um bom momento para revisá-la.
  3. Um convite nunca pode ser editado. caso seja necessário, peça ao coordenador ou ministerial para reprová-lo e faça um novo convite com os dados corretos.
  4. Todos os dados da vaga aparecerão para os possíveis aprovadores e para o estdante. Apenas os comentários que você fez sobre o estudante são estritamente confidenciais.
  5. Você só poderá fazer outro convite para essa vaga caso ele não seja aprovado ou caso o estudante rejeite.
